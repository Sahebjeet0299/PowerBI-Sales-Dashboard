#### **# Power BI Sales Dashboard**



\## Objective

Create an interactive sales dashboard to analyze revenue trends across cities, products, and time periods.



\## Tools Used

\- Power BI

\- CSV Dataset



\## Dataset Information

The dataset contains:

\- Order ID

\- Date

\- City

\- Product

\- Sales Amount



\## Dashboard Features

\- Total Sales KPI

\- Sales by City

\- Sales by Product

\- Monthly Sales Trend

\- Interactive City Slicer



\## Key Insights

\- Bangalore generated the highest revenue.

\- Laptop category contributed the most sales.

\- Sales remained stable across first 2 months,then dropped.

\- Interactive filtering allows city-wise analysis.



\## Files Included

\- sales\_dashboard.pbix → Power BI dashboard

\- sales.csv → raw dataset
